__Level-up - Aplicacion movil__

Nuestra aplicación móvil está desarrollada en Kotlin + Jetpack Compose, conectada a un backend en Spring Boot, diseñada para ofrecer una experiencia de compra moderna y fluida para productos gamer.
Incluye manejo de usuarios, productos, carrito, boletas, y opiniones.

__A) Utilizamos las tecnologias:__

__Frontend:__

  Kotlin,
  Jetpack Compose,
  MVVM,
  Retrofit,
  Coroutines.

__Backend:__

  Spring Boot,
  JWT Security,
  PostgreSQL,
  Maven,
  AWS.

Otros:
GitHub,
Trello.


__B) Estructura del proyecto__

  ui/ → Pantallas

  
  viewmodel/ → Lógica de negocio

  
  remote/ → DTO + Retrofit

  
  repository/ → Fuente de datos

  
  core/ → Utilidades, sesión, interceptor

__C) Instalación__

__Para Android:__

1. Clona el repositorio
   
  git clone https://github.com/RafaelRiosFredes/LevelupApp.git
  
2. Abre Android Studio
 
3. Sincroniza Gradle
 
4. Ejecuta en un emulador Android

__Para Backend:__

1. Clona el repositorio
   
git clone https://github.com/RafaelRiosFredes/LevelUpBackend.git

   
2. Abre con IntelliJ IDEA

   
3. Corre el proyecto desde IntelliJ


__D) Funcionalidades principales:__


1.- Registro de usuarios.

2.- Login con JWT.

3.- Listado de productos.

4.- Detalle de producto.

5.- Opiniones y valoraciones.

6.- Carrito de compras.

7.- Generación y visualización de boletas.

8.- Roles.
